# Adaptation Spectrum

- [Adaptation Spectrum](./adaptation.md) — The ladder of options to steer a base model, ordered by cost & how much you change weights.
- [Chain-of-Thought](./cot.md) — Prompt the model to reason step-by-step before answering — boosts reasoning accuracy.
- [Continued Pretraining](./continued-pretraining.md) — More self-supervised pretraining on domain text (law, code, medicine) before any SFT.
- [Few-Shot / ICL](./few-shot.md) — Put labeled examples in the prompt; the model infers the pattern (in-context learning).
- [Full Fine-Tuning](./full-ft.md) — Update ALL weights on your data. Maximum capacity, maximum cost & risk.
- [Prompting](./prompting.md) — Steer behavior purely via the input text — zero weight changes, instant iteration.
- [RAG](./rag.md) — Retrieve relevant documents and inject them into the prompt — adds knowledge without training.
- [ReAct / Tool Use](./react.md) — Interleave reasoning with tool/API calls — the basis of agents.
